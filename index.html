Important

In arabic you almost never write short vowels. For example قطة [qiṭa] - cat.
We can see ق [q], ط [ṭ] and ة [a]. The "i" here is a short vowel so we skip it
in writing. Type qṭä in the converter if you want to get this exact word.
Another example: فلسطين [filasṭīn] - Palestine. Only the second "i" is long,
so we write it like flsṭin.
